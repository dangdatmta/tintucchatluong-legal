Tin Tức Chất Lượng

Privacy Policy

Effective: January 1, 2025  ·  Last updated: May 2025
This Application does not collect, sell, or share personal data from any user. It exists solely to post video content to the account owner's social media profiles via authorized APIs.

1. Who We Are

Tin Tức Chất Lượng is a personal content automation project that publishes daily news summary videos to social media platforms including TikTok, YouTube, and Facebook. The Application is not a commercial product and is not made available to the general public.

2. What Data We Access

To function, the Application accesses the following through TikTok's official API:

  • The account owner's TikTok Creator Info (display name, privacy level options, posting limits) — used solely to validate posting eligibility before each upload.
  • An OAuth access token and refresh token — used to authenticate API requests on behalf of the account owner.

No data from TikTok users who view or interact with published content is ever collected, stored, or processed by this Application.

3. What Data We Do Not Collect

  • No names, emails, or contact details of any individual
  • No browsing history, device identifiers, or IP addresses
  • No analytics or behavioral data from platform audiences
  • No data from minors or any third-party users

4. How Credentials Are Stored

API credentials (client keys, secrets, and refresh tokens) are stored exclusively as encrypted secrets in GitHub Actions, the CI/CD environment used to run the Application. These values are never logged, printed, or transmitted to any service other than the official API endpoints of TikTok, YouTube, and Facebook.

5. Data Sharing

We do not sell, rent, lease, or otherwise share any data with any third party. The Application communicates only with the official API endpoints of the respective social media platforms.

6. Cookies & Tracking

This Application is not a website and does not use cookies, tracking pixels, or any analytics tools. This policy page itself does not embed any tracking scripts.

7. Your Rights

As the Application does not collect personal data from any individual other than the account owner, there is no personal data subject to deletion, portability, or correction requests. If you have concerns about how TikTok handles your data as a platform user, please refer to TikTok's Privacy Policy.

8. Changes to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected on this page with an updated effective date.

Contact
For any privacy-related questions, please reach out via the Tin Tức Chất Lượng TikTok account inbox.
